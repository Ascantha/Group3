TicTacToe README

Launch Instructions:
1. Install JavaFX library on the computer to be used
2. Run the command 'javac TicTacToe.java'
3. Run the command java TicTacToe

Game Manual
In the game, the player can face off against either a human or computer opponent. The opponent has three difficulty settings, 
the hardest of which is impossible to beat, only draw. In either selection of opponent, a number of matches to be played must
be selected to start. Once in the game, selecting a space can be done by simply clicking on the desired space. To go back to
the main menu, click the "back" button in the top right corner of the screeen.
